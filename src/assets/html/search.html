What am I searching?

NIST Library Search contains ~300,000 titles owned by NIST, along with hundreds of millions of electronic resources such as journal and newspaper articles, book chapters, ebooks, conference proceedings, videos, reports, reviews, and more. Most of the content from the individual databases is discoverable in NIST Library Search.

The Default "Everything" Search

The default "Everything" search is a blended search of the other search profiles (Library Catalog, Articles, and NIST Technical Series).

If expanded, millions of resources NIST doesn't own are displayed, which allows a broader search and the ability to request resources through Interlibrary Loan (ILL).

Specific Search Scopes

Library Catalog Search

Searching the Library Catalog displays print and electronic books, print and electronic journals (not including articles), and physical and electronic audio/visual material.

Articles Search

Searching Articles displays journal and newspaper articles, books chapters, reviews, and preprints.

NIST Technical Series Search

Searching NIST Technical Series limits the search to NIST Technical Series Publications.
